We are proud to announce Gatito Linux, the distro that refuses to adopt the modern, heavy software. Written primarily in Assembly, Gatito Linux aims to keep everything simple. That means **any x86 device could theoretically run it.**

Featuring gatinit, a minimalistic initialization system that does nothing but power your OS. No journal, no daemon, and no telemetry. The system comes with nothing beyond its core. It's up to you to choose your Window Manager, File Explorer and Audio Server.

Gatito Linux ships with gatloader, which aims to be as bare-bones as possible. No UEFI support, and no multi-boot. You can change it to GRUB or Limine if you need to.

No coreutils. Gatito Linux uses ASMUtils, which is essentially coreutils rewritten in Assembly, but with zero dependencies and an extremely optimized codebase.

In addition, Gatito comes with some extra tools, including a custom browser and shell. These features are experimental and may be discontinued in the future.

**Build the system as you wish. Keep internet free.**
